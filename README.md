# TaskBoard - Task Management App

A modern, role-aware task management board with Kanban workflows, invitations, and a refined dashboard experience. Built with Vite + React + TypeScript and styled with Tailwind + shadcn/ui.

**Live demo:** https://task-board-task-management-app.vercel.app/

---

## Features

- Kanban board with drag-and-drop task and column reordering
- Role-based permissions (owner, manager, contributor, viewer)
- Invitation system (send/accept/decline) with inbox panel
- Insight dashboards for manager and employee views
- Workspace management (create, archive, delete)
- Polished settings page (profile, security, theme)

---

## Tech Stack

- **Frontend:** Vite, React, TypeScript
- **State:** Zustand
- **UI:** Tailwind CSS, shadcn/ui, lucide-react
- **Drag & Drop:** @dnd-kit
- **Charts:** Recharts
- **Routing:** react-router-dom
